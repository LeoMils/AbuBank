══════════════════════════════════════
BUBBLE DESIGN — CRITICAL
══════════════════════════════════════

Shape: perfect circle. NOT rounded-square. NOT squircle.

Feel: "quiet luxury bubble"
  - softly spherical
  - matte-glass, NOT glossy-glass
  - subtle inner highlight at top (very restrained)
  - subtle rim light
  - soft drop shadow
  - slight dimensional depth
  - NO chrome / NO neon glow / NO sci-fi

Background inside bubble: per-service color (see LOGOS section above)
Logo inside: <img> tag, object-fit: contain, padding 14%, centered

Hover/tap feedback:
  - scale: 1.04 on press
  - shadow slightly deeper
  - NOT bouncy, NOT playful

CSS target:
  border-radius: 50%
  background: [per-service color]
  box-shadow: 0 4px 18px rgba(0,0,0,0.45), inset 0 1px 0 rgba(255,255,255,0.12)
  overflow: hidden
