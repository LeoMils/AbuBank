# Rule: Senior-First UX

All UI decisions must prioritize an 80+ non-technical user.

- Touch targets: minimum 48×48px, recommended 56px.
- Text: minimum 16px body, 18-20px for interactive elements.
- Contrast: 4.5:1 minimum for text on dark backgrounds.
- No scroll on primary screens (Home, Calendar main view, Settings collapsed, Voice mode).
- No color-only indicators — always pair with shape/text.
- No complex gestures — one-tap for all critical actions.
- Error messages in plain Hebrew, never technical jargon.
- Always show what state the app is in (listening, thinking, speaking, error).
- Vibration feedback (15ms) on key actions where supported.
- When in doubt, remove complexity rather than add explanation.
