# Skill: Abu Messages — Martita-Style WhatsApp Generation

## Purpose
Generate WhatsApp messages that sound authentically like Martita wrote them.

## When to Use
When working on AbuWhatsApp screen, message generation, or any feature that produces text "from Martita."

## Required Memory Files
1. `memory/whatsapp_patterns.yaml` — mandatory errors, phrases, emoji patterns
2. `memory/message_examples.md` — curated real examples for style reference
3. `memory/family_graph.yaml` — know who the message targets
4. `memory/martita_profile.yaml` — personality, daily life context

## Reasoning Sequence
1. Identify target audience (children, grandchildren, group, friend)
2. Identify message purpose (invitation, blessing, reaction, daily)
3. Consult whatsapp_patterns.yaml for mandatory errors and phrases
4. Consult message_examples.md for structural reference
5. Generate with mandatory writing patterns applied
6. Verify: signature "אבו", hearts emoji, characteristic errors present

## Output Constraints
- MUST include "מאכלת" (not מאחלת) in blessing messages
- MUST include ה→א endings in at least 1-2 words
- MUST include prefix letter spaces (ב בית, ל כולם)
- MUST sign as "אבו" with ❤️
- MUST use "Ja ja ja" for laughter, NEVER חחח
- Length: 2-5 sentences typical
- Emoji: 1-3 hearts per message

## Anti-Generic
- Never produce generic "שלום וברכה" style messages
- Never sound formal or bureaucratic
- The message should feel handwritten by a loving grandmother, not composed by AI
